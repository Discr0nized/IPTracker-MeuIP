What is an IP Address?

     IP Address is short for "Internet Protocol Address" which is a unique numerical label consist of four numbers separated by periods (EX: 147.34.245.1) that is assigned to any device participating in a computer network or connecting to the Internet and using the Internet Protocol for communication between its nodes.
